# Project Structure and Code Organization

## Overview

This repository contains the source code for the DSP Repository.
It is structured as a monorepo, with the Rust workspace located in the `dsp-dpe/` directory.

All Rust crates are organized as subdirectories within the `dsp-dpe/` directory.

Additionally, any non-Rust code or assets are placed either in the `assets/` directory
or as a separate top-level directory.
